Here's what SkyWay is and what it does:

**SkyWay** is a dark-themed, mobile-first travel booking app for flights and trains, built as a single HTML file you can open in any browser or deploy as a web app.

**What it covers:**

It has six complete screens that flow into each other — Home, Search Results, Flight/Train Detail, Traveller Details (booking form), Payment, and Booking Confirmation. Every screen has smooth slide transitions, a sticky status bar with a live clock, and a bottom navigation bar.

On the **Home screen** you get a dual-mode search card that switches between flights and trains. You can tap city codes to pick from a list, select departure and return dates from a scrollable date picker, choose cabin class (Economy, Business, First), and tap popular route tiles for a quick search.

The **Results screen** shows real flight listings (IndiGo, Air India, Vistara, SpiceJet) with prices, duration, stop count, amenity tags like WiFi and meals, and discount percentages. Train results show IRCTC-style data — train number, name, type, all class options (SL, 3A, 2A, 1A, CC), availability status, and waitlist counts.

The **Detail screen** for flights includes a full interactive seat map where you can tap to select available seats, with premium rows marked in gold. For trains it shows class selection across six tiers and berth preference (Lower, Middle, Upper, etc.).

The **Booking screen** has passenger count controls for adults and children, a full traveller form (name, DOB, gender, passport), contact details, optional GST input for business bookings, and a live price breakdown with taxes and fees.

The **Payment screen** offers UPI, credit/debit card, net banking, wallets, and no-cost EMI. There's a promo code input with one-click preset codes, and a processing animation when you confirm.

The **Confirmation screen** generates a styled ticket with a PNR code, a barcode strip, all journey details, and download/home buttons. The "My Trips" tab in the nav also shows your last booked ticket.

The design uses a deep near-black background (`#0a0a0f`), purple as the primary accent for flights, teal for trains, and the Syne typeface for all headings to give it that 3D-minimal, premium feel.
